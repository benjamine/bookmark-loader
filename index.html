Bookmark-Loader

Load a menu of bookmarks on your page using a browser bookmarklet. Bookmarklet Inception!

Instructions

  • Drag this bookmarklet: Bookmarks to your browser bookmarks toolbar
  • Edit the bookmarklet url replacing the .md url with your own bookmarks file (you can change the bookmark title accordingly)
  • Start using your bookmarklet!

Bookmarks file

To use your own bookmarks file it has to be available at a domain that supports CORS. You can host it on github and access it using rawgithub.com

The bookmarks file is markdown document containing a list of links. Each list items can contain text (useful to group links), or an hyperlink.
